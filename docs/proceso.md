# 📋 Proceso de Adopción

Adoptar una mascota es un acto de amor que cambia dos vidas: la tuya y la de ella. Aquí te explicamos paso a paso cómo hacerlo de forma responsable y segura. 🌷

---

!!! success "¡Adoptar es la mejor opción!"
    Al adoptar en lugar de comprar, le das una segunda oportunidad a una vida que te lo agradecerá con amor incondicional cada día de su vida. 🐾

---

## 🗺️ Pasos del proceso

| # | Paso | Descripción | Tiempo estimado |
|---|---|---|---|
| **1** | 📝 Solicitud | Llenar el formulario en línea | 10 minutos |
| **2** | 💬 Entrevista | Reunión con el equipo del refugio | 1–2 días hábiles |
| **3** | 🐾 Encuentro | Visita al refugio y conoce a la mascota | 1 día |
| **4** | 🏠 Evaluación | Revisión del hogar (fotos o visita) | 2–3 días hábiles |
| **5** | 📄 Contrato | Firma del acuerdo de adopción responsable | 1 día |
| **6** | 🎉 ¡A casa! | Tu nueva mascota llega a su hogar | ¡El mejor día! |

---

## 📝 Requisitos para adoptar

=== "✅ Requisitos generales"

    ```
    ┌──────────────────────────────────────────────┐
    │  REQUISITOS PARA ADOPTAR                     │
    │──────────────────────────────────────────────│
    │  ✅  Ser mayor de 18 años                    │
    │  ✅  Presentar identificación oficial        │
    │  ✅  Firmar contrato de adopción responsable │
    │  ✅  Comprometerse a no abandonar ni maltratar│
    │  ✅  Brindar atención veterinaria básica      │
    │  ✅  Proporcionar alimentación y agua fresca  │
    └──────────────────────────────────────────────┘
    ```

=== "🐶 Para perros"

    ```python
    # Checklist de adopción canina
    requisitos = {
        "Espacio":     "Patio o acceso a área de paseos",
        "Tiempo":      "Mínimo 2 horas de atención al día",
        "Paseos":      "Al menos 2 veces por día",
        "Vacunas":     "Refuerzo anual obligatorio",
        "Veterinario": "Visita mínima 1 vez al año",
        "Alimentación":"Comida balanceada según tamaño",
    }

    print("🐶 Requisitos para adoptar un perro:\n")
    for item, detalle in requisitos.items():
        print(f"  → {item}: {detalle}")
    ```

=== "🐱 Para gatos"

    ```python
    # Checklist de adopción felina
    requisitos = {
        "Espacio":     "Apartamento o casa (espacio interior)",
        "Tiempo":      "Al menos 1 hora de juego diario",
        "Arenero":     "Limpieza cada 2 días como mínimo",
        "Vacunas":     "Refuerzo anual obligatorio",
        "Veterinario": "Visita mínima 1 vez al año",
        "Alimentación":"Croquetas de calidad + agua fresca",
    }

    print("🐱 Requisitos para adoptar un gato:\n")
    for item, detalle in requisitos.items():
        print(f"  → {item}: {detalle}")
    ```

## 💰 Costos del proceso

| Concepto | Detalle | Costo |
|---|---|---|
| Proceso de adopción | Gestión y seguimiento completo | **Gratuito** 🎁 |
| Vacunas al día | Incluidas antes de la entrega | **Sin costo** ✅ |
| Esterilización | Incluida si la mascota lo requiere | **Sin costo** ✅ |
| Microchip de identificación | Incluido en todas las adopciones | **Sin costo** ✅ |
| Donación voluntaria | Para apoyar al refugio | A tu criterio 💚 |

!!! tip "¿Cómo apoyar al refugio?"
    Aunque la adopción es completamente gratuita, las donaciones nos permiten seguir rescatando y cuidando más animales. ¡Cualquier aporte, grande o pequeño, suma muchísimo! 🌸

---

## 🚫 Causas de rechazo

!!! warning "Ten esto en cuenta"
    Una solicitud puede ser rechazada si detectamos alguna de las siguientes situaciones:

    - Historial comprobado de maltrato o abandono animal.
    - Vivienda que no permite mascotas sin alternativa viable presentada.
    - Incapacidad de comprometerse con los cuidados básicos requeridos.
    - Información falsa o incompleta en el formulario de solicitud.
